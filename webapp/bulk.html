Import / Export Data

In this tab you can import export data easily to the app with a csv file. Most fitness apps offer a bulk export functionality to a csv file so you can use this file to transfer the data from apps like Strava,Garmin connect,Polar flow...

Unfortunately all those apps offer a different csv format export so you should customize it first in order Leading pace to import it succesfully.

Export from Leading Pace

Click the download button. All your activity data will appear in a comma seperated value form on the screen. Copy and Paste this data to a blank txt file on your device. As this app is in an early stage and changes and lost of data my happen it is adviced to do this regurarly in order to not lose your data. The format of the exported data is the same as the format Leading Pace accepts as an import. 

  Import to Leading Pace

Format your activity data in the below format.

date,duration,distance,hr,AvSpeed,runningIndex,up,down,end
11/26/2019,27.17,3920.0,171,8.7,40,0,0,0
11/26/2019,27.17,3920.0,171,8.7,40,0,0,0

Caution:
1)date format should be mm/dd/year seperated with "/" like above

2)duration is in minutes
3)distance in meters
4)every column should be named exactly like above
5)the last column should me "end"

Save the txt file and then import it with the import function above.
